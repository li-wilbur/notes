# 如何发布新笔记

本文档将指导你如何在个人笔记站点中添加和发布新的文章。

## 1. 创建 Markdown 文件

在 `docs` 目录下的对应分类文件夹中创建一个新的 `.md` 文件。

例如，如果你想写一篇关于 **TypeScript** 的技术笔记：
1. 进入 `docs/技术笔记/前端开发/` 目录。
2. 创建文件 `TypeScript入门.md`。
3. 编写内容：

```markdown
# TypeScript 入门

TypeScript 是 JavaScript 的超集...

## 基础类型

- boolean
- number
- string
...
```

## 2. 配置侧边栏 (Sidebar)

为了让新文章在左侧导航栏显示，你需要修改配置文件。

1. 打开 `docs/.vitepress/config.mts` 文件。
2. 找到 `sidebar` 配置项。
3. 在对应的分类下添加新文件的路径：

```typescript
'/技术笔记/': [
  {
    text: '前端开发',
    collapsed: false,
    items: [
      { text: 'React 核心概念', link: '/技术笔记/前端开发/React核心概念' },
      // 新增下面这一行
      { text: 'TypeScript 入门', link: '/技术笔记/前端开发/TypeScript入门' }
    ]
  }
],
```

## 3. 本地预览

在终端运行以下命令，确保新页面显示正常：

```bash
npm run docs:dev
```

访问 `http://localhost:5173/notes/` 查看效果。

## 4. 部署上线

将更改推送到 GitHub，等待 Action 自动构建部署：

```bash
git add .
git commit -m "feat: add new article about typescript"
git push
```

稍等几分钟，访问你的 GitHub Pages 地址即可看到更新。
