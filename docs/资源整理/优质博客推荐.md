# 优质博客推荐

## 前端技术

1. [Dan Abramov's Blog](https://overreacted.io/) - React 核心团队成员的博客，深入浅出。
2. [CSS-Tricks](https://css-tricks.com/) - CSS 技巧和教程的宝库。
3. [MDN Web Docs](https://developer.mozilla.org/) - Web 开发最权威的参考文档。

## 后端与架构

1. [Martin Fowler](https://martinfowler.com/) - 软件架构大师，微服务、重构等概念的提出者。
2. [High Scalability](http://highscalability.com/) - 涵盖各大互联网公司的架构案例。

## 综合

1. [Hacker News](https://news.ycombinator.com/) - 极客新闻聚合。
2. [GitHub Explore](https://github.com/explore) - 发现有趣的开源项目。
