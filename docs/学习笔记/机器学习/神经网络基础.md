# 神经网络基础

## 神经元模型

人工神经元是神经网络的基本单元，模拟生物神经元的工作方式。

$$ y = f(\sum_{i=1}^{n} w_i x_i + b) $$

其中：
- $x_i$: 输入信号
- $w_i$: 权重
- $b$: 偏置
- $f$: 激活函数

## 常见激活函数

1. **Sigmoid**: 将输入映射到 (0, 1) 区间。
2. **ReLU**: $f(x) = \max(0, x)$，目前最常用。
3. **Tanh**: 将输入映射到 (-1, 1) 区间。

## 反向传播算法

通过计算损失函数相对于权重的梯度，利用链式法则更新权重，以最小化预测误差。
